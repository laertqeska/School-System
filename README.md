# Multi-School Management System

A Spring Boot backend application for managing multiple schools, including students, teachers, courses, and classes. The project is designed with a layered architecture to ensure maintainable and scalable code.

## Technologies Used

- Java 21
- Spring Boot
- PostgreSQL
- RESTful APIs
- Git / GitHub
- Spring Security (role-based authentication)

## Features

- REST APIs for managing multiple schools, students, teachers, and classes
- Role-based authentication using Spring Security
- Layered architecture (Controller, Service, Repository)
- **Soft delete** for entities
- Optimized queries to fix **N+1 query problems**
